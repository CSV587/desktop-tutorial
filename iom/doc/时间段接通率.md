## 时间段接通率API

##### 通讯方式：
```
* 类型：POST 
* content-type:application/json
```

##### request：//时间段接通率 请求
```
{
    projectName:"test",//项目名称
    queryMode:"1",//1按年查询 2按月查询 3按日查询
    date:"2019"//如果按日查询格式为yyyy-mm-dd 如果按月查询格式为yyyy-mm 如果按年查询则为yyyy
}
```

##### response：//时间段接通率 响应
```
{
  "code":0,
  "message":"操作成功",
  "time":"2018-07-06 11:16:25",
  "data":{ //第一个元素是一次外呼接通率 第二个元素是多次外呼接通率
    '9:00':[0.2,0.9],
    '10:00':[0.6,0.8],
    '11:00':[0.7,0.7],
    '12:00':[0.54,0.8],
    '13:00':[0.89,0.9],
    '14:00':[0.3,0.8],
    '15:00':[0.1,0.9]
  },
  "success":true
}
```
