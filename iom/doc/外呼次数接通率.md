## 外呼次数接通率API

##### 通讯方式：
```
* 类型：POST 
* content-type:application/json
```

##### request：//外呼次数接通率 请求
```
{
    projectName:"test",//项目名称
    queryMode:"1",//1按年查询 2按月查询
    date:"2019"//如果按月查询格式为yyyy-mm 如果按年查询则为yyyy
}
```

##### response：//外呼次数接通率 响应
```
{
  "code":0,
  "message":"操作成功",
  "time":"2018-07-06 11:16:25",
  "data":{ //第一个元素是一次外呼接通率 第二个元素是多次外呼接通率
    '1月':[0.2,0.9],
    '2月':[0.6,0.8],
    '3月':[0.7,0.7],
    '4月':[0.54,0.8],
    '5月':[0.89,0.9],
    '6月':[0.3,0.8],
    '7月':[0.1,0.9],
    '8月':[0.91,0.09],
    '9月':[0.1,0.9],
    '10月':[0.3,0.7],
    '11月':[0.1,0.9],
    '1月':[0.7,0.3]
  },
  "success":true
}
```
