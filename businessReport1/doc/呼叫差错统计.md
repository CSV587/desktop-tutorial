## 呼叫差错统计API

##### 通讯方式：
```
* 类型：POST 
* content-proType:application/json
```

##### request_1：//呼叫差错统计 请求
```
{
    projectName:"test",//项目名称
    startDate:"2018-01-01",//外呼开始时间 上限
    endDate:"2018-01-05"//外呼结束时间 下限
}
```

##### response_1：//呼叫差错统计 响应
```
{
  "code":0,
  "message":"操作成功",
  "time":"2018-07-06 11:16:25",
  "data":{
    detail:[
      {name:'转写失败',proType:"transError",value:445},
      {name:'匹配失败',proType:"matchError",value:134},
      {name:'异常中断',proType:"disconnectError",value:200},
      {name:'通话成功',proType:"connectSuccess",value:1548}
    ],
    sum:[
      {name:'通话差错',value:779},
      {name:'通话成功',value:1548}
    ]
  },
  "success":true
}
```

##### request_2：//呼叫差错详细统计 请求
```
{
    projectName:"test",//项目名称
    startDate:"2018-01-01",//外呼开始时间 上限
    endDate:"2018-01-05",//外呼结束时间 下限
	proType:"transError", //取自response_1 type里的值
	pagination:{	//分页参数
		 current: 1,
         pageSize: 10
	}
}
```
