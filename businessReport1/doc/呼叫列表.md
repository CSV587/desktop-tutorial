## 呼叫列表API

##### 通讯方式：
```
* 类型：POST 
* content-proType:application/json
```

##### request：//更新 callResult更改为用户输入（输入框） onState：传值更改为connect和disconnect
```
{
   projectName:"test",//项目名称
   recordStartTime:"2018-07-01",//开始时间
   recordEndTime:"2018-08-01",//结束时间
   callResult:"核身,用户挂机",//直接传送呼叫结果名称
   callnumber:"18326485085",//被叫号码
   onState:"connect",//接通状态 connect:接通 disconnect:未接通
   callCount:"1",//外呼次数 1:一次呼叫，2:多次呼叫
   durationStart:0,//时长上限
   durationEnd:50,  //时长下限
   pagination:{	//分页参数
	 current: 1,
     pageSize: 10
   }
}
```
